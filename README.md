# dental-clinic-timeslot-api-flask

The code is generated by:
```
swagger_py_codegen -s swagger.yaml timeslotAPI --ui --spec
```

## Get started using Docker container

Build the Docker image:
```
docker build -t timeslot-api .
```

Run the Docker container:
```
docker run -p 8084:5002 -t timeslot-api __init__.py
```

The GUI of the API is at:
```
http://0.0.0.0:8084/static/swagger-ui/index.html#/
```
(Note: the exposed container port 5002 is mapped to the local machine port 8084)

Use the API from:
```
http://0.0.0.0:8084/v1/timeslots
http://0.0.0.0:8084/v1/timeslots?dentist=Dr+Green&startTime=10
http://0.0.0.0:8084/v1/timeslots/2
```
